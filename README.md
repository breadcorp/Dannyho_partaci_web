# Dannyho Parťáci

Web application for managing convoys and community events for the Dannyho Parťáci VTC(aready closed)

The system allows drivers and partner VTCs to view upcoming convoys, register convoy slots, and access information about events organized by the community.

## Features
- Upcoming convoy overview
- Convoy event database
- Slot booking system for VTCs
- Partner and team information
- Login system for managing convoy data

## About the Community
Dannyho Parťáci is a Czech-Slovak-Polish virtual trucking company (VTC) founded in 2024.  
The community focuses on realistic trucking gameplay, organized convoys, and community events within the ETS2 / TruckersMP ecosystem. :contentReference[oaicite:1]{index=1}

## Use Case
The project is designed to simplify convoy organization and event coordination for VTC communities.
